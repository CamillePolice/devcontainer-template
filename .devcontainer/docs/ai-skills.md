# AI Agent Skills

Enhance your AI coding assistants with reusable capabilities from [Skills.sh](https://skills.sh).

## 📖 What Are Skills?

Skills are reusable capabilities for AI agents that provide procedural knowledge and best practices. They work with **Cursor**, **Claude Code**, **GitHub Copilot**, and many other AI assistants.

---

## 🚀 Installation

Install skills with a single command:

```bash
npx skills add <owner/repo>
```
